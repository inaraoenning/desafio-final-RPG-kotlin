    ⚔️ Como Jogar (Regras da Arena)

    1. Decidam no chat quem será o HOST (Cria a sala) e quem será o CLIENTE.

    2. O HOST deve olhar os IDs disponíveis e criar a batalha no Passo 1 abaixo. O ID gerado aparecerá no chat.

    3. O CLIENTE vai para o Passo 2, coloca o ID da batalha, escolhe sua ação (Atacar/Defender/Poder) e clica em "Enviar Ação". O Host não saberá o que o cliente escolheu.

    4. O HOST vai para o Passo 2, escolhe a sua ação e clica em "Executar Turno". Os dados rolam, os atributos são calculados e o resultado aparece para os dois!
